# Connexta — Site Institucional 2025

Site institucional completo desenvolvido como Trabalho de Conclusão de Curso em Desenvolvimento de Sistemas.
Instituição: Etec Professor Horácio Augusto da Silveira.  
Ano: 2025

Site institucional completo, moderno e totalmente responsivo da **Connexta** — empresa de inovação aberta que conecta corporações e startups com foco no ODS 9 da ONU.

Link ao vivo: https://joelumiri.github.io/site_institucional_Connexta/

## Características principais

- Design dark premium com rosa neon (#FF007F)  
- Partículas animadas na home (particles.js)  
- Animações suaves com AOS  
- Scrollbar personalizada rosa  
- Formulário de contato funcional (via Formspree)  
- Totalmente responsivo (mobile first)  
- SEO otimizado  
- Favicon personalizado (CX rosa com fundo transparente)  
- Estrutura limpa com HTML, CSS e JS separados por página  

## Tecnologias usadas

- HTML5 + CSS3 (sem frameworks)
- particles.js (home)
- AOS Library (animações)
- Formspree (formulário)
- Google Fonts: Inter + Satoshi
